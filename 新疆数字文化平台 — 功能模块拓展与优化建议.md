## Profile 重设计全量 ToDo（含扩展项）

1. **备份与准备**
   - 备份 `frontend/src/views/Profile.vue`。
   - 确认本地依赖已安装，`pnpm run dev` 可正常启动。

2. **主题变量与全局样式**
   - 在 `src/styles/theme-variables.css` 定义 `--theme-primary`、`--theme-accent`、`--bg-paper`、`--card-shadow`、`--radius-md` 等。
   - 在主入口（`main.ts`/`main.js`）全局引入该样式文件。

3. **新增前端组件**
   - `ScenicHero.vue`：风景背景 + avatar + CTA。
   - `GalleryCarousel.vue`：用户图片画廊（支持懒加载）。
   - `MiniEventsCalendar.vue`：小型活动日历列表。
   - 可选：占位图、默认 `default-scenic.jpg` 等静态资源。

4. **Profile.vue 重构**
   - 引入并注册新组件。
   - 用 `<ScenicHero>` 替换旧 hero 卡片。
   - 在 Hero 下方新增 `<GalleryCarousel>` + `<MiniEventsCalendar>` 布局区块。
   - 更新样式以使用 CSS 变量，统一背景、圆角、阴影、行距。

5. **数据层对齐**
   - 确保 `userInfo`/store 提供 `cover`、`gallery`、`registeredEvents`、`badges` 等字段；前端对缺失数据提供默认值。
   - 标记后端接口变更需求（行程列表、活动报名、分享导出等）。

6. **交互与可访问性**
   - 完善 i18n 文案（Hero 按钮、日历标题等）。
   - 处理图片懒加载/占位符。
   - 补充 ARIA、键盘导航、按钮语义化。

7. **验证与分阶段提交**
   - 本地联调新布局、移动端响应式、浅色/深色配色适配。
   - 按组件新增、页面改造、样式主题等拆分 commit。

8. **扩展功能（建议分阶段实施）**
   - **附近体验/地图模块**：在 Profile 中嵌入地图（Leaflet/Mapbox），展示附近景点或向导。
   - **成就徽章体系**：新增徽章组件，展示贡献值、任务解锁逻辑及后端统计。
   - **文化故事/藏品发布**：集成富文本编辑器（tiptap/quill），支持图文、地理标签、图片画廊。
   - **社交分享与导出**：提供微信/微博分享按钮，增加行程导出为 PDF/图片功能（后端短链或前端 jsPDF）。
   - **多语言与少数民族语言切换**：在 Profile 显示语言偏好，提供自动切换入口（如维语、哈萨克语）。
   - **互动微动效与反馈**：统一保存、分享、报名等动作的 0.15–0.25s 动效，卡片 hover 阴影提升。
   - **活动/行程收藏联动**：在 Profile 展示“我的行程”“收藏体验”，可跳转或分享。
   - **进阶统计面板**：可视化用户足迹（地图热力）、互动次数、收藏类型等。

9. **测试与交付**
   - 功能自测、视觉验收、无障碍检查。
   - 根据扩展范围同步后端接口文档、联调计划。
